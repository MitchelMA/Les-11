Hoe gebruik je de class selector in HTML en CSS?

Maak een lege map aan

Hiermee maak je een lege map aan.

README.md

Zet een README.md bestand in deze map.

Init

open de terminal, en navigeer naar deze map. Vervolgens typ je "git init" om een git repository aan te maken in deze map.

Tracker

Track README.md door "git add README.md" in de terminal te typen.

Branch

Voeg een branch toe aan je git repository met "git branch -M main", waar "main" de naam van de branch is.

GitHub

Maak nu een repository aan op GitHub.

Remote

Voeg een remote aan je git toe genaamd origin met “git remote add origin {jouw GitHub repository.git}.

Commit

Commit de veranderingen met git commit -m "commit_message".

Push

Push je git repository naar github met “git push -u origin main”.
